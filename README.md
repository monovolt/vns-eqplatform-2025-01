# vns-eqplatform-2025-01


아두이노 핀 연결

0 : TMC2226 제어 - RX
1 : TMC2226 제어 - TX


| 핀 이름            | 설명                                    |
| --------------- | ------------------------------------- |
| **DIR**         | 방향 제어                                 |
| **STEP**        | 스텝 신호                                 |
| **NC**          | 사용 안 함                                |
| **UART** (첫 번째) | **TX → 드라이버 입력용** (보통 Arduino TX와 연결) |
| **UART** (두 번째) | **RX ← 드라이버 출력용** (보통 Arduino RX와 연결) |
| **MS2, MS1**    | 마이크로스텝 설정                             |
| **EN**          | Enable 핀 (LOW일 때 활성화)                 |
