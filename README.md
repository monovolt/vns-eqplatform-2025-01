# vns-eqplatform-2025-01


## 아두이노 핀 연결

### 디지털 핀

| 아두이노 핀 | 연결 대상 | 용도 | 비고 |
|-----------|---------|------|------|
| **0 (RX)** | TMC2209 UART TX | 스텝 드라이버 통신 | Serial1 RX |
| **1 (TX)** | TMC2209 UART RX | 스텝 드라이버 통신 | Serial1 TX |
| **2** | TMC2209 EN | 모터 Enable | LOW = 활성화 |
| **3** | TMC2209 STEP | 스텝 펄스 신호 | - |
| **4** | TMC2209 DIR | 방향 제어 | HIGH = 시계, LOW = 반시계 |
| **5** | 버튼 1 | 시작 위치 확인 | INPUT_PULLUP |
| **6** | 버튼 2 | 종료 위치 확인 | INPUT_PULLUP |
| **7** | 버튼 3 | 모터 정지 | INPUT_PULLUP |
| **8** | 버튼 4 | 원래 방향 재구동 | INPUT_PULLUP |
| **9** | 버튼 5 | 수평 레벨링 | INPUT_PULLUP |
| **10** | Piezo 스피커 | 오디오 피드백 | - |
| **11** | Red LED | 보조배터리 유지용 | 항상 HIGH |
| **12** | IR 리시버 | 적외선 리모컨 수신 | - |

### I2C 핀

| 아두이노 핀 | 연결 대상 | 용도 | 주소 |
|-----------|---------|------|------|
| **SDA** | OLED SSD1306, MPU-9250 | I2C 데이터 | 0x3C (OLED) |
| **SCL** | OLED SSD1306, MPU-9250 | I2C 클럭 | - |

### TMC2209 스텝 드라이버 연결

| TMC2209 핀 | 아두이노 핀 | 설명 |
|-----------|-----------|------|
| **DIR** | 4번 핀 | 방향 제어 |
| **STEP** | 3번 핀 | 스텝 신호 |
| **EN** | 2번 핀 | Enable (LOW = 활성화) |
| **UART TX** | 0번 핀 (RX) | 드라이버 → Arduino |
| **UART RX** | 1번 핀 (TX) | Arduino → 드라이버 |
| **MS2, MS1** | - | 마이크로스텝 설정 (코드에서 8 microsteps 설정) |
| **NC** | - | 사용 안 함 |





## 본체 버튼

| 버튼 번호 | 핀 번호 | 기능 | 상태 변경 |
|---------|--------|------|----------|
| **버튼 1** | 5번 핀 | 오른쪽 끝 확인 (시작 위치 도달) | → `Start Rdy` |
| **버튼 2** | 6번 핀 | 왼쪽 끝 확인 (종료 위치 도달) | → `End.` |
| **버튼 3** | 7번 핀 | 모터 정지 | → `Stopped` |
| **버튼 4** | 8번 핀 | 원래 방향 재구동 | → `Tracking..` 또는 `Mov B Pos` |
| **버튼 5** | 9번 핀 | 수평 레벨링 (가운데로 이동) | → `Leveling>` / `Leveling<` → `Welcome..` |

## IR 리모컨 버튼

### 숫자 버튼
| 버튼 | 코드 | 기능 | 비고 |
|-----|------|------|------|
| **1** | 0x45 (69) | 종료 위치 이동 완료 | → `End.` + LG 종료음 |
| **2** | 0x46 (70) | 모터 정지 | → `Stopped` |
| **3** | 0x47 (71) | 시작 위치 이동 완료 | → `Start Rdy` + 반짝반짝 작은별 |
| **4** | 0x44 (68) | 속도 감소 (500씩) | tracking_delay += 500 |
| **5** | 0x40 (64) | 트래킹 시작 | → `Tracking..` (어떤 상태에서든) |
| **6** | 0x43 (67) | 속도 증가 (500씩) | tracking_delay -= 500 |
| **7** | 0x07 (7) | 소리 ON/OFF 토글 | ON시 짧은 beep 확인음 |
| **8** | 0x15 (21) | 센서 ON/OFF 토글 | ON시 높은음, OFF시 낮은음 |
| **9** | 0x09 (9) | **센서 영점 보정 저장** | `Welcome..` 상태 & ±3도 이내에서만 작동 |
| **0** | 0x19 (25) | 현재 속도 저장 | `Tracking..` 상태일 때만 저장 가능 |
| **\*** | 0x16 (22) | 수평 레벨링 (센터로 이동) | → `Leveling>` / `Leveling<` → `Welcome..` |
| **#** | 0x0D (13) | 모터 정지 | 버튼 2와 동일 |

### 방향 버튼
| 버튼 | 코드 | 기능 | 비고 |
|-----|------|------|------|
| **↑** | 0x18 (24) | 속도 미세 증가 (20씩) | tracking_delay -= 20 |
| **↓** | 0x52 (82) | 속도 미세 감소 (20씩) | tracking_delay += 20 |
| **→** | 0x5A (90) | 시작 위치로 빠르게 이동 | → `Mov B Pos` (반시계 방향) |
| **←** | 0x08 (8) | 종료 위치로 빠르게 이동 | → `Mov E Pos` (시계 방향) |

### 기타 버튼
| 버튼 | 코드 | 기능 | 비고 |
|-----|------|------|------|
| **OK** | 0x1C (28) | 저장된 속도로 재구동 | default_delay로 리셋 후 구동 |

## 상태(Status) 설명

| 상태 | 의미 | 모터 동작 |
|------|------|----------|
| `Welcome..` | 초기 상태 / 수평 상태 | 정지 |
| `Start Rdy` | 시작 위치 준비 완료 (오른쪽 끝) | 정지 |
| `Tracking..` | 천체 추적 중 | 느린 속도 (시계방향) |
| `End.` | 추적 종료 (왼쪽 끝 도달) | 정지 |
| `Stopped` | 사용자 정지 | 정지 |
| `Mov B Pos` | 시작 위치로 복귀 중 | 빠른 속도 (반시계방향) |
| `Mov E Pos` | 종료 위치로 이동 중 | 빠른 속도 (시계방향) |
| `Leveling>` | 우측으로 수평 조정 중 | 빠른 속도 (반시계방향) |
| `Leveling<` | 좌측으로 수평 조정 중 | 빠른 속도 (시계방향) |

## 센서 영점 보정 방법

1. 버블 수평계로 물리적 수평 맞춤
2. 본체 버튼 5 또는 IR `*` 버튼으로 `Welcome..` 상태 진입
3. IR 버튼 `9` 누름
4. 성공음(빠른 3음) 또는 실패음(낮은 beep) 확인
5. OLED 화면에서 `L/R Angle: 0.0 (보정값)` 형식으로 offset 확인

**주의:** 센서가 OFF 상태이거나, Welcome 상태가 아니거나, ±3도 이상 기울어진 경우 저장되지 않습니다.
